# LHABA-project

# Proyecto Liga de Handball de Buenos Aires (rama femenina)

# Aplicacion desarrollada para la gestion de clubes, equipos y planteles que competiran en la liga

Funcionalidades:
.-Creacion de nuevos clubes
.-Creacion de nuevos miembros de clubes
.-Visualizacion de informacion de los clubes, equipos y planteles

Instalacion:
.-Clonar repositorio
    git clone https://github.com/Belen-Bustillo/LHABA-project.git
    cd \path\al\proyecto

.-Crear entorno virtual
    python -m venv venv
    venv\Scripts\activate

.-Instalar paquetes necesarios
    pip install -r requirements.txt

.-Ejecutar servidor
    python manage.py runserver

.-Ingresar desde navegador web
    http://127.0.0.1:8000/
